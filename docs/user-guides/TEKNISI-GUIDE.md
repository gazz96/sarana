## üë®‚Äçüîß Handling Repairs

### Accept Problem Assignment
1. View problem details
2. Review the issue and affected items
3. Click **"Accept Assignment"**
4. Status changes to "PROSES" (In Progress)

### Start Working on Problem
1. Go to **Assigned Problems**
2. Find the assigned problem
3. Click **"Start Work"**
4. Status updates to show you're actively working

### Update Problem Status
1. Open problem details
2. Click **"Update Status"**
3. Select current status:
   - **PROSES**: Actively working on repair
   - **SELESAI DIKERJAKAN**: Work completed
4. Add notes about progress
5. Click **"Update"**

### Add Repair Notes
1. In problem details, find **"Add Notes"**
2. Write about:
   - What you found when inspecting
   - Parts needed (if any)
   - Estimated completion time
   - Any complications encountered
3. Click **"Save Notes"**

### Complete Repair
1. Finish the repair work
2. Click **"Complete Repair"**
3. Fill in completion details:
   - **Actual Cost**: Final repair cost
   - **Work Done**: Description of repair
   - **Parts Used**: Any replacement parts
   - **Time Spent**: Hours worked
4. Click **"Mark as Completed"**
5. Status changes to "SELESAI DIKERJAKAN"

### Report Issues with Problem
1. If you can't complete the repair:
2. Click **"Report Issue"**
3. Explain the problem:
   - Parts not available
   - Need specialized equipment
   - Requires different expertise
   - Location inaccessible
4. Submit for admin review

## üìã Managing Problem Items

### View Affected Items
1. Open problem details
2. See **"Problem Items"** section
3. For each item:
   - Item name and code
   - Quantity affected
   - Estimated cost
   - Damage description
   - Photos (if available)

### Update Item Status
1. In problem details, find item list
2. Click **"Edit"** next to item
3. Update information:
   - **Actual Cost**: Real repair cost
   - **Repair Details**: What you fixed
   - **Parts Replaced**: Any new parts used
4. Click **"Update Item"**

### Upload Repair Photos
1. In problem details, click **"Add Photos"**
2. Upload photos showing:
   - Before repair (damage)
   - During repair (work in progress)
   - After repair (fixed item)
3. Add descriptions to each photo
4. Click **"Upload"**

## üõ†Ô∏è Technical Procedures

### Repair Workflow
```
Assigned ‚Üí Accept ‚Üí Start Work ‚Üí Repair ‚Üí Complete ‚Üí Report
```

### Priority Guidelines
- **High Priority**: Safety issues, major facility damage
- **Medium Priority**: Learning disruptions, discomfort
- **Low Priority**: Minor issues, cosmetic damage

### Time Management
- **Respond to Assignment**: Within 1 hour
- **Start Work**: Within 4 hours of assignment
- **Complete Repairs**: Based on severity
  * High Priority: Same day
  * Medium Priority: Within 2 days
  * Low Priority: Within 1 week

## üö´ Problem Reporting

### When to Report Issues
1. **Missing Parts**: Required parts not available
2. **Safety Concerns**: Unsafe working conditions
3. **Need Help**: Problem requires additional expertise
4. **Access Issues**: Can't reach repair location
5. **Wrong Assignment**: Problem outside your expertise

### How to Report
1. Click **"Report Issue"** in problem
2. Choose issue type
3. Provide detailed explanation
4. Suggest solution if possible
5. Submit to admin
